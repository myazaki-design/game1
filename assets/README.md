# アセット

## 音声
- **gold.mp3** - 収穫でコイン獲得時に再生
- **shop.mp3** - 種購入・畑アンロック時に再生

- **Seedling_002.png** - 成長中（後半）の苗

- **harvest_002.png** - にんじんの収穫時
- **harvest_003.png** - とうもろこしの収穫時
- **home_001.png** - お家（120×120推奨）
- **map_obj_001.png** - 木（りんごの木）
- **map_obj_002.png** - 草
- **map_obj_003.png** - 花（白）
- **map_obj_004.png** - 石
- **map_obj_005.png** - 花（黄）
- **map_obj_006.png** - 木（常緑）
